# 女人在想什么

**副标题：不是猜心术，而是亲密关系中的情绪、需求与误解**

这是一部关于亲密关系的心理写作书稿。它并不试图提供一份粗暴的“女性说明书”，而是从安全感、位置感、自我价值感、依恋模式、表达、倾听、冲突与修复等角度，理解女性在爱中的感受逻辑，也帮助读者理解亲密关系本身。
