# 女人在想什么

**副标题：不是猜心术，而是亲密关系中的情绪、需求与误解**

这是一部关于亲密关系的心理写作书稿。它并不试图提供一份粗暴的“女性说明书”，而是从安全感、位置感、自我价值感、依恋模式、表达、倾听、冲突与修复等角度，理解女性在爱中的感受逻辑，也帮助读者理解亲密关系本身。

## 六部分目录

### 第一部分：她不是难懂，她是在用另一种方式表达
1. 她说的不是字面意思，而是关系里的感受
2. 她真正想知道的，常常不是答案，而是态度
3. 她为什么会敏感

### 第二部分：她真正渴望的，不只是爱，而是被安放
4. 安全感：她不是想控制你，她是怕失去
5. 归属感：她想要的不是“有个人”，而是“被放在心上”
6. 尊重感：很多女人不是败给不爱，而是败给被轻慢

### 第三部分：她在爱你，也在带着过去靠近你
7. 依恋模式：她的爱，可能带着过去的影子
8. 投射：你爱的未必是她，她爱的也未必是你
9. 自我价值感：她在要爱的同时，也在确认自己值不值得被爱

### 第四部分：关系真正变坏，常常不是因为大事，而是因为一次次没被接住
10. 她不是在吵那件事，她是在吵“我是不是不重要”
11. 冷战：她为什么不说
12. 失望的形成：爱不是突然没的，是一点点漏光的

### 第五部分：真正让关系走远的，不只是喜欢，而是能力
13. 倾听：不是等她说完，而是听懂她没说出口的部分
14. 表达：成熟男人不是会哄，是会说真话
15. 修复：关系里最重要的能力，不是避免冲突，而是修复冲突

### 第六部分：看懂她，也是在看懂你自己
16. 你以为她难，其实是你没学会亲密
17. 好的关系，不是猜中心思，而是一起把心思说明白

## 扉页文案

> 你以为你想懂女人，其实你是不想再被关系困住。
